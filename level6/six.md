# Activity 6: Positioning Elements

**Objective:** Learn how to position elements using Tailwind's positioning utilities.

**Tasks:**
1. Add `relative` to the second `<div>` element (white card container) - makes it position relative
2. Add `absolute top-4 right-4` to the `<span>` element (badge) - positions it in top-right corner
3. Add `fixed bottom-0 left-0 right-0` to the last `<div>` element (footer) - fixes it at bottom
4. The badge should already have `top-4 right-4` from step 2
5. Add `z-10` to the `<span>` element (badge) - ensures it appears above other elements

**Key Concepts:**
- `relative`, `absolute`, `fixed` for positioning
- `top-{size}`, `right-{size}`, `bottom-{size}`, `left-{size}` for positioning
- `z-{number}` for z-index layering
