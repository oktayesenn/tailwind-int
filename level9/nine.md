# Activity 9: Responsive Design

**Objective:** Master Tailwind's responsive breakpoints for mobile-first design.

**Tasks:**
1. Add `text-base md:text-xl lg:text-2xl` to the `<h1>` element - responsive text sizing
2. Add `hidden md:block` to the first `<div>` element (sidebar) - hidden on mobile, visible on md+
3. Add `grid grid-cols-1 md:grid-cols-3 gap-4` to the second `<div>` element (card container) - responsive grid
4. Add `p-4 md:p-8` to the `<body>` element - responsive padding
5. Add `text-center md:text-left` to the last `<p>` element - responsive text alignment

**Key Concepts:**
- `sm:`, `md:`, `lg:`, `xl:`, `2xl:` breakpoint prefixes
- Mobile-first approach (base styles for mobile)
- Responsive utilities for layout changes
- `hidden` and `block` for visibility control
