# Activity 8: Hover and Focus States

**Objective:** Learn how to style interactive states with Tailwind.

**Tasks:**
1. Add `hover:bg-blue-600` to the `<button>` element - changes background on hover
2. Add `hover:scale-105` to the `<button>` element - scales up slightly on hover
3. Add `focus:ring-2 focus:ring-blue-500 focus:outline-none` to the `<input>` element - adds focus ring
4. Add `transition-all duration-300` to the `<button>` element - makes animations smooth
5. Add `active:scale-95` to the `<a>` element (green button) - scales down on click

**Key Concepts:**
- `hover:{utility}` for hover states
- `focus:{utility}` for focus states
- `active:{utility}` for active/click states
- `transition-{property}` for animations
- `duration-{time}` for transition duration
- `scale-{size}` for transform scaling
