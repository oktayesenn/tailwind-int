# Activity 7: Width and Height Utilities

**Objective:** Master Tailwind's width and height utilities for sizing elements.

**Tasks:**
1. Add `max-w-md mx-auto` to the outer `<div>` element (white container) - sets max-width and centers it
2. No image in this activity, but `w-full` would make an image full width
3. Add `w-64` to the second `<div>` element (sidebar) - sets width to 256px
4. Add `h-screen` to the `<body>` element - makes it full viewport height
5. Add `min-h-96` to the third `<div>` element (content area) - ensures minimum height

**Key Concepts:**
- `w-{size}` for width (w-full, w-64, w-1/2)
- `h-{size}` for height (h-screen, h-full, h-96)
- `max-w-{size}` for maximum width
- `min-h-{size}` for minimum height
- `mx-auto` for horizontal centering
