# Activity 1: Basic Spacing and Colors

**Objective:** Learn Tailwind's spacing and color utilities.

**Tasks:**
1. Add `p-4` to the `<div>` element (main container) - adds 16px padding to all sides
2. Add `bg-blue-500` to the `<div>` element - sets background color to blue-500
3. Add `mt-6` to the `<h1>` element - adds 24px margin-top to the heading
4. Add `text-white` to the `<h1>` element - makes the text color white
5. Add `px-2 py-1` to the `<button>` element - adds horizontal padding of 8px and vertical padding of 4px

**Key Concepts:**
- `p-{size}` for padding (p-4 = 16px)
- `m-{size}` for margin (m-6 = 24px)
- `bg-{color}-{shade}` for background colors
- `text-{color}` for text colors
- `px-{size}` and `py-{size}` for directional spacing
