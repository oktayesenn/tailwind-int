# Activity 12: Navigation Bar

**Objective:** Create a responsive navigation bar using Flexbox.

**Tasks:**
1. Add `flex justify-between` to the inner `<div>` element inside `<nav>` - creates flex container with space between
2. Add `items-center` to the inner `<div>` element inside `<nav>` - vertically centers items
3. Add `hover:text-blue-600` to all `<a>` elements (navigation links) - hover color change
4. Add `py-4 px-6` to the `<nav>` element - adds padding to navbar
5. Add `md:hidden` to the `<button>` element (Menu button) - hides on md screens, visible on mobile

**Key Concepts:**
- Building navigation components
- Flexbox for navbar layouts
- Responsive navigation patterns
- Hover states for links
