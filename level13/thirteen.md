# Activity 13: Form Styling

**Objective:** Style form elements with Tailwind utilities.

**Tasks:**
1. Add `border-2 border-gray-300 rounded-lg px-4 py-2` to both `<input>` elements and the `<textarea>` element - styles form fields
2. Add `focus:border-blue-500 focus:outline-none` to both `<input>` elements and the `<textarea>` element - focus states
3. Add `text-sm font-medium text-gray-700` to all `<label>` elements - styles labels
4. Add `border-red-500` (in addition to existing classes) to the email `<input>` element - error state example
5. Add `w-full md:w-auto` to the `<button>` element - full width on mobile, auto on md+

**Key Concepts:**
- Form element styling
- Focus states for inputs
- Error state handling
- Responsive form layouts
