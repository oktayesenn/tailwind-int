# Activity 10: Building a Button Component

**Objective:** Create a reusable button component using Tailwind utilities.

**Tasks:**
1. Add `bg-blue-500 text-white rounded-lg` to the first `<button>` element (Primary Button) - base button styles
2. Add `px-6 py-3` to the first `<button>` element - adds padding
3. Add `hover:bg-blue-600 transition` to the first `<button>` element - hover effect and smooth transition
4. Add `focus:outline-none focus:ring-2 focus:ring-blue-500` to the first `<button>` element - focus state
5. Add `inline-flex items-center` to the first `<button>` element - allows icon alignment

**Key Concepts:**
- Combining multiple utilities for component styling
- Creating consistent button styles
- Using inline-flex for icon alignment
- Building reusable patterns
