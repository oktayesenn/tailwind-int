# Activity 5: Grid Layout

**Objective:** Learn Tailwind's grid system for creating complex layouts.

**Tasks:**
1. Add `grid` to the outer `<div>` element (container) - creates a grid container
2. Add `grid-cols-3` to the outer `<div>` element - makes it a 3-column grid
3. Add `gap-4` to the outer `<div>` element - adds spacing between grid items
4. Add `col-span-2` to the first inner `<div>` element (Item 1) - makes it span 2 columns
5. Add `col-span-2` to the last inner `<div>` element (Item 5) - makes it span 2 columns

**Key Concepts:**
- `grid` to create a grid container
- `grid-cols-{number}` for column count
- `grid-rows-{number}` for row count
- `col-span-{number}` to span columns
- `gap-{size}` for grid gaps
