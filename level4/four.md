# Activity 4: Flexbox Layout

**Objective:** Master Tailwind's flexbox utilities for creating flexible layouts.

**Tasks:**
1. Add `flex` to the outer `<div>` element (container) - makes it a flex container
2. Add `justify-center` to the outer `<div>` element - centers items horizontally
3. Add `items-center` to the outer `<div>` element - centers items vertically
4. Add `gap-4` to the outer `<div>` element - adds spacing between flex items
5. Add `flex-1` to the middle `<div>` element (Item 2) - makes it take remaining space

**Key Concepts:**
- `flex` to create a flex container
- `justify-{position}` for horizontal alignment (justify-center, justify-between)
- `items-{position}` for vertical alignment (items-center)
- `gap-{size}` for spacing between flex items
- `flex-{size}` to control flex grow
